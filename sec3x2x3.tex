\subsection{逆関数の微分法}
\label{sec3x2x3}

$y=f(x)$を区間$I$で定義された$x$の連続な単調関数としたとき，
その逆関数$x=f^{-1}(y)$は，定理2.7により，区間$f(I)$で定義された
$y$の連続な単調関数である．このとき

\定理{3.4}
区間$I$で$y=f(x)$が$x$について微分可能で$f'(x) \ne 0$ならば，
$x=f^{-1}(y)$は区間$f(I)$で$y$について微分可能であって
\begin{equation}
 \frac{d x}{d y} = 1 \bigg/ \left( \frac{dy}{dx} \right).
																											\label{eq3x16}
\end{equation}
\証明
$y$の増分$\varDelta y$に対応する$x$の増分を$\varDelta x$とすれば，
$f^{-1} (y)$が連続で単調であるから，$\varDelta y \rightarrow 0$のとき
$\varDelta x \rightarrow 0$で$\varDelta y \ne 0$のとき
$\varDelta x \ne 0$である．故に
\[
	\frac{d x}{d y} = \lim_{\varDelta\rightarrow 0} \frac{\varDelta x}{\varDelta y}
                  = \lim_{\varDelta \rightarrow0} \left( 1 \bigg/ \left(  \frac{\varDelta y}{\varDelta x} \right) \right)
                  = 1 \bigg/ \left( \frac{d y}{d x}\right).
\]
\■

$y=f(x)$が連続な単調増加関数であっても或る点$x$で$f'(x) = 0$となることは
しばしば起こる．例えば$f(x) = x^3$とおけば
$\displaystyle f'(0) = \lim_{\varDelta x \rightarrow 0} 
(\varDelta x)^3 / \varDelta x = \lim_{\varDelta x \rightarrow 0} (\varDelta x)^2
=0$．このとき$\varDelta y = (\varDelta x)^3$であるから
\begin{figure}[hbtp]
	\centering
	\includegraphics[bb=0 0 800 200,clip,width=6cm,height=6cm]{p115.bmp}
\end{figure}
\[
	\lim_{\varDelta y\rightarrow 0} \frac{\varDelta x}{\varDelta y} 
= \lim_{\varDelta x\rightarrow 0} \frac{1}{(\varDelta x)^2} = + \infty,
\]
すなわち$f^{-1} (y) = y^{1/3}$は$y=0$で微分可能ではない．
