\subsection{一様収束と連続性}
\label{sec5x3x2}

\定理{5.5}
関数$f_n	(x)	,	n	=1,	2,	3,	\cdots,$はすべて区間$I$で連続であるとする．
このとき，
\begin{list}{}{}
	\item	$(1^\circ)$	関数列$\{f_n(x)	\}$が$I$で一様に収束すれば，
その極限$\displaystyle	f(x)	=	\lim_{n\rightarrow\infty}	f_n	(x)$も$I$で連続である．
	\item	$(2^\circ)$	級数$\displaystyle	\sum_{n=1}^{\infty}	f_n	(x)$が$I$で一様に収束すれば，
その和：$\displaystyle	s	(x)	=	\sum_{n=1}^{\infty}	f_n	(x)$も$I$で連続である．
\end{list}
\証明
$(2^\circ)$は$(1^\circ)$の系であるから$(1^\circ)$を証明すればよい．
このためには$I$に属する各点$a$において$f(x)$が連続であることをいえばよいから，
点$a	\in	I$を一つ定めて考える．任意の正の実数$\varepsilon$に対応して，仮定により，
自然数$n_0(	\varepsilon	)$が定まって，すべての点$x	\in	I$において
\[
	n	>	n_0(	\varepsilon)	\quad	\text{ならば}	\quad	|	f_n(x)	-	f(x)	|	<	\varepsilon
\]
となる．また，$f_n	(	x)$は$a$で連続であるから，正の実数$\delta_n	(\varepsilon)$が定まって
\[
	|	x-a	|	<	\delta_n	(	\varepsilon	)	\quad	\text{ならば}	\quad	|f_n	(x)	-	f_n(a)	|	<	\varepsilon
\]
となる．そこで，まず	$n	>	n_0	(\varepsilon)$なる自然数$n$を一つ定め，つぎに$\delta	(	\varepsilon)	=	\delta_n	(	\varepsilon)$
とおく．
\[
	|	f(x)	-	f(a)	|	\leqq	|	f(x)	-	f_n	(x)	|	+	|	f_n(x)	-	f_n	(a)	|	+	|	f_n	(a)	-	f(a)	|
\]
であるから
\[
	|	f(x)	-	f(a)	|	<	2	\varepsilon	+	|	f_n(x)	-	f_n(a)	|.
\]
故に
\[
	|x	-	a|	<	\delta(\varepsilon	)	\quad	\text{ならば}	\quad	|f(x)	-	f(a)	|	<	3\varepsilon
\]
となるが，$\varepsilon$は任意の正の実数であった．故に$f(x)	$は$a$で連続である．
\■

各項$f_n(x)$が区間$I$で連続な関数列$\{f_n(x)	\}$が$I$で収束するけれども一様に収束しない場合には，
上記の例5.4が示すように，極限$f(x)	\displaystyle	=	\lim_{n\rightarrow\infty}	f_n(x)$は必ずしも
$I$で連続でない．

\例{5.5}
各自然数$n$に対し区間$[0,3]$で連続な関数$f_n(x)$を次のように定義する：
\[
	\begin{cases}
		\displaystyle	\:	0						\leqq	x	\leqq	\frac{1}{n}	\quad	\text{のとき}	\quad	f_n(x)	=	nx,	\\
		\:	\\
		\displaystyle	\:	\frac{1}{n}	\leqq	x	\leqq	\frac{2}{n}	\quad	\text{のとき}	\quad	f_n(x)	=	2	-	nx,	\\
		\:	\\
		\displaystyle	\:	\frac{2}{n}	\leqq	x	\leqq	3						\quad	\text{のとき}	\quad	f_n(x)	=	0.
	\end{cases}
\]
\begin{figure}[hbtp]
	\centering
	\includegraphics[bb=0 0 800 200,clip,width=74.4mm,height=40.1mm]{p219.bmp}
\end{figure}
まず$f_n	(0)	=	0$である．また$x$，$0	<	x	\leqq	3$，を任意に与えたとき，$\displaystyle	n	>	2/x$ならば
$f_n(x)	=	0$であるから，$\displaystyle	\lim_{n\rightarrow\infty}	f_n(x)	=	0$．
故に区間$[0,3]$で関数列$\{f_n(x)\}$は収束し，
極限$\displaystyle	0	=	\lim_{n\rightarrow\infty}	f_n(x)$は連続である．
しかし，$f_n(1/n)=1$であるから，$\varepsilon	<	1$の場合は，$	|f_n(1/n)	-	f(1/n)|	>	\varepsilon$となってしまい，
収束は$[0,3]$で一様でない．

\定理{5.6}	級数$\displaystyle	\sum_{n=1}^{\infty}	f_n(x)$を収束する正項級数
$\displaystyle	\sum_{n=1}^{\infty}	a_n$と比較したとき，区間$I$でつねに$|f_n(x)	|	\leqq	a_n$
であったとする．このとき
\begin{list}{}{}
	\item	($1^\circ$)	級数$\displaystyle	\sum_{n=1}^{\infty}	f_n	(x)	$は区間$I$で\underline{一様に絶対収束する}．
	\item	($2^\circ$)	各項$f_n	(x)$が区間$I$で連続ならば，和$\displaystyle	s(x)	=	\sum_{n=1}^{\infty}	f_n	(x)$も区間
$I$で連続である．
\end{list}
\証明
\[
	\sigma_{n,m}	(x)	=	|	f_{n+1}	(x)	|	+	|	f_{n+2}(x)	|	+	\cdots	+	|	f_m	(x)	|
\]
とおけば，区間$I$で常に
\[
	\sigma_{n,m}	(x)	\leqq	a_{n+1}	+	a_{n+2}	+	\cdots	+	a_m
\]
であるから，Cauchyの判定法（定理 5.4）により，$\displaystyle	\sum_{n=1}^{\infty}	f_n(x)	$は
$I$で一様に絶対収束する．したがって$\displaystyle	\sum_{n=1}^{\infty}	f_n(x)	$は$I$で
一様に収束するから，各項$f_n(x)$が$I$で連続ならば，
定理5.5，$(2^\circ)$により，$s(x)	=	\displaystyle	\sum_{n=1}^{\infty}	f_n(x)$も区間$I$で連続である．
\■

単調な連続関数列の収束に関してはつぎの定理が成り立つ：

\定理{5.7}（Dini の定理）
閉区間$[a,b]$で\underline{連続な}関数$f_n(x)$を項とする単調非増加な関数列，すなわち
\[
	f_1	(x)	\geqq	f_2	(x)	\geqq	f_3(x)	\geqq	\cdots
					\geqq	\cdots
					\geqq	f_n(x)	\geqq	\cdots,	\quad	a	\leqq	x	\leqq	b,
\]
なる関数列$\{f_n(x)\}$が$[a,b]$で\underline{連続な}関数$f	(x)$に収束するならば，
$\{f_n(x)\}$は$[a,b]$で\underline{一様に}$f(x)$に収束する．

\証明
$g_n	(x)	=	f_n(x)	-	f(x)$とおいて関数列$\{g_n(x)\}$が
$[a,b]$で一様に$0$に収束することを証明すればよい．仮定により，
区間$[a,b]$で$g_n(x)$は連続，関数列$\{g_n	(x)	\}$は単調非増加で（各点で）$0$に
収束する．この収束が$[a,b]$で一様でなかったとすれば，或る正の実数
$\varepsilon_0$に対しては，どんな自然数$n$をとっても，$m	>	n$のとき
$[a,b]$でつねに$g_m	(	x)	<	\varepsilon_0$とはならない．すなわち，各自然数
$n$に対して$g_m(c_n)	\geqq	\varepsilon_0$なる自然数$m,m	>	n$，と点$c_n	,	a	\leqq	c_n	\leqq	b$，
が存在する．
\footnote{
各$g_n$に対応して少なくとも，ある1点$c_n$で突き抜けている．
}
このとき仮定より，$g_n	(c_n)	\geqq	g_m	(c_n)$であるから，
\begin{equation}
	g_n(c_n)	\geqq	\varepsilon_0.
\label{eq5x24}
\end{equation}
この$c_n$のなす点列$\{	c_n	\}$は，\S	\ref{sec1x6x5}，定理1.30により，
収束する部分列$c_{n_1},	c_{n_2},	c_{n_2},	\cdots	,	c_{n_j},	\cdots$をもつ
その極限を$	c	=\displaystyle	\lim_{j\rightarrow\infty}	c_{n_j}$とすれば，
$g_n	(x)	$は$x$について連続であるから，
\[
	g_n	(c)	=	\lim_{j\rightarrow\infty}	g_n	(c_{n_j})
\]
であるが，$n_j	>	n$ならば，（\ref{eq5x24}）と仮定により，
\[
	g_n	(c_{n_j})	\geqq	g_{n_j}	(c_{n_j})	\geqq	\varepsilon_0,
\]
\footnote{
	$c_{n_j}		\in	\{	c_n	\}$は（\ref{eq5x24}）の性質を持つ集合の１つの元であった．
}
したがって
\[
	g_n	(c)	\geqq	\varepsilon_0	>	0.
\]
これは関数列$\{	g_n	(x)	\}$が（各点で）$0$に収束することに矛盾する．故に
$\{g_n	(x)	\}$は$[a,b]$で一様に$0$に収束する．
\■
